# QA Automation Project

This project demonstrates QA automation using Selenium and Python.

## Features

- Page Object Model implemented
- Automated form submission test on DemoQA
- Assertions to verify form data
- Uses pytest for testing
- Works with WebDriver Manager for automatic ChromeDriver handling

## Requirements

- Python 3.x
- Selenium
- WebDriver Manager
- Pytest
